# Schedule
慧虹课题组以物联网（IoT）与人工智能（AI）**技术为核心研究方向，致力于推动工业系统的智能化、网络化与自主化发展。团队的研究重点聚焦于智能交通、风电能源等典型工业场景，围绕感知计算、时空数据建模、智能决策与大模型推理等关键问题，探索数据驱动与机理约束相结合的智能运维新范式。
目前，团队承担国家重点研发计划等多项科研任务，聚焦智能运维与人工智能交叉领域的关键科学问题与工程应用，主要研究方向包括：
- **数据分布质量核查与一致性评估**
> 面向复杂工业数据的分布质量与一致性问题，构建可量化的数据可靠性分析体系。重点研究缺失数据的分布恢复与一致性重建，以提升数据的完整性与统计代表性。
  
- **基于大模型的知识图谱构建**
> 探索语义理解与结构化知识融合机制,重点实现知识图谱辅助大模型输出的协同构建与双阶段推理过程，提升模型的知识完整性与推理一致性。

- **基于强化学习的大模型深度推理与风电并网控制决策优化**
> 面向复杂工况下的多目标决策问题，构建具有自适应权衡能力的智能控制策略。重点关注多目标强化学习的决策优化与可解释性机制，以提升模型在安全性、稳定性与透明性方面的可信度。

- **时序特征驱动的大模型体系设计与训练机制研究**
> 探索时序动态特征与模型结构的协同学习机制，以提升大模型在预测、推理与泛化方面的时序感知能力。
